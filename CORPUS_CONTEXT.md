# Corpus Context

This repository sits on the book layer of a wider public corpus. It is not a theory repository by itself. It is a reader-facing publication repository for Volume II of Qubit of Hope.

Operationally, this repository:

- preserves publication assets
- makes provenance explicit
- documents drift instead of erasing it
- carries visible integrity manifests
- remains legible to humans and tools

A book repository is not a decorative wrapper. It is a stable container for files, paths, provenance, and citation.
